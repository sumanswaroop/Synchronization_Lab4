Setup:
	
	1) HP Laptops with following specs:
	Server:
	Ram : 8GB, i7 Processor
	Client: 
	Ram : 8GB, i5 Processor




1)To run:
make

./server <args>
./client <args>


2)Detailed Analysis of output is present in plots folder along with graph.

Case1.eps : Two graphs in the same file.

case1.txt has the following format

<number of workers> <Throughput> <Avg Res time> 

Case2.eps : Three graphs in the same file.
case2.txt has the following format

<number of workers> <Throughput> <Avg Res time> <Number of denied requests>
